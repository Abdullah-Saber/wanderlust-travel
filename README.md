# Wanderlust - Tourism Landing Page

A modern, high-converting tourism landing page built with HTML, CSS, and JavaScript.

## Features

- **Hero Section** - Immersive background with search functionality
- **Popular Destinations** - 5 curated travel destinations with quick view
- **Experience Cards** - Adventure, Culture, and Relaxation packages
- **Testimonials** - Auto-rotating carousel with user reviews
- **Call-to-Action** - Conversion-focused booking section
- **Responsive Design** - Mobile-first approach

## Tech Stack

- Semantic HTML5
- CSS3 with Custom Properties
- Vanilla JavaScript (ES6+)
- No external dependencies

## Quick Start

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project
cd wanderlust-travel

# Open in browser
open index.html
```

## Deployment

### Option 1: Netlify (Recommended)

1. Go to [Netlify](https://app.netlify.com/)
2. Drag and drop the project folder
3. Site will be live instantly

### Option 2: GitHub Pages

1. Push to GitHub
2. Enable GitHub Pages in repository settings
3. Select main branch as source

### Option 3: Vercel

1. Go to [Vercel](https://vercel.com/)
2. Import GitHub repository
3. Deploy with default settings
